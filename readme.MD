<h1 align="center">Creador de Personajes D&D 5e</h1>
Esta es una aplicación de Python diseñada para hacer más fácil la creación 
de fichas de personaje de Dungeons & Dragons (5ª Edición). 
El objetivo principal es ofrecer una herramienta que gestione 
las reglas y cálculos del juego de forma automática, 
para que el usuario no tenga que esforzarse en buscar en manuales ni 
en calcular él mismo las estadísticas.

<h2>Funcionamiento</h2>
El programa se conecta a D&D 5e API, la base de datos oficial de DND para obtener toda la
información necesaria. A través de la terminal, el usuario selecciona lo esencial para su personaje:
su raza, su clase, sus conjuros y su equipo inicial, etc. ...

Al mismo tiempo que el usuario va rellenando su información, la aplicación va guardando sus stats y calculándolas
dependiendo de lo elegido. Por ejemplo, si el usuario elige una raza que aumenta la fuerza, 
el programa actualiza ese valor y sus modificadores de ataque.
Esto asegura que la ficha sea correcta según las reglas del juego.

<h2>¿Para quién?</h2>
Está dirigida principalmente a jugadores nuevos que aún no dominan
la creación de personajes, y para servir de apoyo a 
Dungeon Masters que necesiten generar fichas rápidamente.

<hr>

Al finalizar el proceso, la aplicación muestra la información en pantalla,
y genera un archivo en Excel. Este archivo sirve como base de datos personal 
donde el usuario puede almacenar todos sus personajes.
